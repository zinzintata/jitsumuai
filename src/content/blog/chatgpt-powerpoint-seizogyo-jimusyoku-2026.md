# ChatGPTでパワーポイントを作る方法3選【製造業・事務職向けコピペOKプロンプト付き2026年版】

「プレゼン資料の作成だけで半日つぶれた」——製造業・事務職の現場でよく聞く話です。この記事では、**プログラミング不要・今日から使える**「ChatGPTでパワーポイントを作る方法」を3つ、実際に使えるプロンプトとともに解説します。

---

## ChatGPTを使えば資料作成の時間が劇的に変わる

ある調査（スマートスライド社、2021年・営業職対象）では、プレゼン資料の作成に最も時間がかかると回答した割合は全体の30.5%にのぼります。1スライド作るのに平均30分かかるとすれば、10枚のスライドで5時間。会議の前日に深夜まで作業した経験がある方も多いでしょう。

ChatGPTを使うと、この「資料作成」の仕事が大きく変わります。具体的には次の3つの段階で使えます。

| 使い方 | 節約できる作業 | 無料版 | Plus |
|-------|------------|-------|------|
| ①構成案・スライド内容を作らせる | アイデア出し・文章作成 | ✅ | ✅ |
| ②VBAマクロでスライドを自動生成 | 手入力・レイアウト作業 | ✅ | ✅ |
| ③.pptxファイルをそのまま生成 | ほぼ全工程 | ❌ | ✅ |

## 方法①：構成案からスライド内容まで一気に作る（無料版OK）

最もシンプルな方法です。ChatGPTに「このプレゼンの構成と各スライードの内容を作って」と伝えるだけで、スライドの骨格が完成します。

### 手順

1. ChatGPTに以下のプロンプトを貼り付ける
2. 出力されたスライド内容をPowerPointに手動でコピペする
3. 見た目はPowerPointで仕上げる

### コピペOKプロンプト（製造業・事務職向け）

**【改善提案・月次報告向け】**
```
あなたは製造現場のリーダーです。
以下の条件でパワーポイントの構成と各スライードの内容を作成してください。

【目的】：月次生産会議での改善提案
【聴衆】：工場長・部長（現場を知っている上司）
【枚数】：8枚以内
【内容】：現状の課題、改善案、効果試算、実施スケジュール

各スライドについて以下を出力してください：
- スライドタイトル
- 箇条書き（3〜5項目）
- ひとこと補足（話す内容のヒント）
```

**【業務効率化・DX提案向け】**
```
あなたはDX推進担当者です。
以下の条件で経営層向けプレゼン資料の構成と内容を作成してください。

【目的】：AI・IT導入の稟議提案
【聴衆】：経営層（コスト・効果を重視）
【枚数】：10枚以内
【伝えたいこと】：現状の非効率、AI導入による効果、コスト・期間の見込み

各スライドについて以下を出力してください：
- スライドタイトル
- 本文（箇条書き3〜5項目）
- グラフや図表のアイデア（あれば）
```

### ポイント

- 「聴衆」「目的」「枚数」を最初に伝えると内容の精度が上がります
- 出力された内容が長すぎる場合は「各スライードを1行で要約して」と追加指示
- 製造業特有の用語（5S・QC・工程管理など）はそのまま書いてOK

## 方法②：VBAマクロでパワーポイントを自動生成する（無料版OK）

ChatGPTに「VBAコードを書いてもらい→PowerPointで実行する」方法です。スライードの見た目まで含めて自動で作れるのが特徴です。

### 手順

1. ChatGPTに「VBAでパワーポイントを作るコードを書いて」と指示
2. 出力されたコードをコピー
3. PowerPointを開き「開発タブ」→「マクロ」→「新規作成」でコードを貼り付け
4. 実行ボタンを押すとスライドが自動生成される

### コピペOKプロンプト

```
PowerPointのVBAマクロを書いてください。
以下の仕様でスライドを自動生成してください。

【スライド構成】
1枚目：タイトル「2026年度 生産性向上計画」、サブタイトル「製造部 ○○」
2枚目：「現状の課題」、箇条書き3項目（①段取り時間が長い②ペーパーワークが多い③情報共有が遅い）
3枚目：「改善策」、箇条書き3項目（①段取り手順の標準化②書類のデジタル化③朝礼でのAI活用）
4枚目：「期待効果」、箇条書き3項目（①段取り時間30%削減②残業月10時間削減③情報共有ロス解消）
5枚目：「実施スケジュール」、表形式（月・施策・担当者の3列、3行分）

【デザイン】
- 背景：白
- タイトル文字色：濃紺（#003366）
- フォント：メイリオ、本文サイズ18pt
```

### 注意点

- 「開発タブ」が表示されていない場合：PowerPointの「オプション」→「リボンのユーザー設定」→「開発」にチェック
- 生成されたコードがうまく動かない場合は「エラーが出ました。修正してください」とそのままChatGPTに伝えれば修正してくれます
- マクロのセキュリティ設定によっては実行できない場合があります（IT部門に確認を）

## 方法③：.pptxファイルをそのまま生成・ダウンロードする（ChatGPT Plus限定）

ChatGPT Plusの「データ分析（Advanced Data Analysis）」機能を使うと、PowerPointファイル（.pptx）をChatGPTが内部で生成して、そのままダウンロードできます。手動コピペが一切不要になる最もラクな方法です。

### 手順

1. ChatGPT Plusにログイン
2. 新しいチャットを開き「データ分析を使う」を選択（または自動で有効になる）
3. 以下のプロンプトを送る
4. 生成されたファイルのダウンロードリンクをクリック

### コピペOKプロンプト

```
python-pptxを使って、以下の仕様でPowerPointファイル（.pptx）を生成してダウンロードできるようにしてください。

【ファイル名】：improvement_proposal_2026.pptx

【スライド構成】
1枚目：タイトル「AI活用による業務効率化提案」、サブタイトル「製造部 2026年4月」
2枚目：タイトル「現状の課題」、本文（箇条書き）：
  ・議事録作成に毎回1〜2時間かかっている
  ・報告書のフォーマットが統一されていない
  ・メール対応に1日1時間以上費やしている
3枚目：タイトル「AI導入による解決策」、本文（箇条書き）：
  ・ChatGPTで議事録を5分で自動作成
  ・報告書テンプレートをAIで標準化
  ・メール下書きをAIで自動生成
4枚目：タイトル「導入効果（試算）」、本文（箇条書き）：
  ・議事録作成：月15時間→2時間に削減
  ・報告書作成：月10時間→3時間に削減
  ・メール対応：月20時間→10時間に削減
5枚目：タイトル「次のステップ」、本文（箇条書き）：
  ・5月：パイロット導入（製造部3名）
  ・6月：効果検証・全部門展開の判断
  ・7月〜：全社展開

【デザイン設定】
- スライドサイズ：ワイド（16:9）
- 背景色：白
- タイトル文字：濃紺、32pt、太字
- 本文文字：黒、18pt
```

### ここが一番ラク

- PowerPointを開く必要すらない
- 「3枚目の内容を変更して再生成して」という追加指示にも対応
- 完成品を会議前にダウンロードするだけ

> **ChatGPT Plusを使っていない方へ：** この方法（方法③）はChatGPT Plusでしか使えません。有料プラン（ChatGPT Plus）に加入することで方法①〜③すべてが使えるようになります。議事録・報告書・プレゼン資料をAIで作れるようになれば、月何時間もの作業時間を削減できます。

## よくある質問

**Q：無料版のChatGPTでも使えますか？**
方法①（構成案作成）と方法②（VBAマクロ）は無料版でも使えます。方法③（.pptxファイル直接生成）はChatGPT Plus限定です。

**Q：生成されたスライードの著作権は誰のものですか？**
ChatGPTの出力物の著作権はユーザーに帰属するとされています（OpenAIの利用規約に基づく）。ただし業務での使用時は社内規定も確認してください。

**Q：日本語フォントが崩れてしまいます**
方法③でファイルを生成した場合、フォントが英語フォントになることがあります。プロンプトに「フォント：メイリオまたはMS ゴシック」と追記するか、生成後にPowerPointで一括変換してください。

**Q：スライード内のグラフや図はどうすればいいですか？**
ChatGPTはテキスト・表・箇条書きは得意ですが、グラフの自動生成は苦手です。「このスライードに棒グラフを追加して」と指示すると、グラフのデータをCSV形式で出力してくれるので、PowerPointに貼り付けて活用できます。
