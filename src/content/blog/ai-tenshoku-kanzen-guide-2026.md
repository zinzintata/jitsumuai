## 2. AIに「任せていい部分」と「人間が判断すべき部分」

転職活動でAIを使う前に、一点だけ確認しておきましょう。

### AIに任せていい部分
- 自分の経験や強みの**言語化・整理**
- 職務経歴書・自己PRの**下書き作成**
- 求人票の**読み解き・分析**
- 面接での想定Q&Aの**作成と回答練習**
- 複数社の情報の**整理とまとめ**

### 人間が最終判断すべき部分
- 「この会社に入社するかどうか」の決断
- 年収・条件交渉の内容
- 「なんとなく違和感がある」という直感
- 会社の雰囲気・社風の最終判断（面接での肌感覚）

AIはあくまで**思考と作業のサポート**です。最終的な判断は必ず自分で行ってください。

### 個人情報の入力には注意

転職活動でAIを使うとき、氏名・連絡先・現在の会社名などの個人情報はなるべく伏せて入力してください。たとえば「私は〇〇製造株式会社に勤めており...」ではなく、「私は中小製造業の品質管理担当で...」と置き換えるだけで、個人が特定されるリスクを大幅に下げられます。

## 3. ステップ1：自己分析——AIと対話して「強み」を言語化する

転職活動で最初につまずくのが自己分析です。「自分の強みって何だろう？」という問いに、一人で向き合い続けるのはしんどい。AIを「壁打ち相手」として使えば、対話の中から自分でも気づいていなかった強みが浮かび上がってきます。

### なぜAIが自己分析に有効か

AIは質問を重ねながら、あなたが何気なく話したエピソードから「それはこういう強みですね」と言語化してくれます。友人や家族への相談と違って、遠慮なく何時間でも付き合ってくれるのが大きな利点です。

### コピペOKプロンプト①：強み発掘

```
あなたはキャリアコーチです。私の転職活動の自己分析を手伝ってください。
以下の情報をもとに、私の強みを3〜5個、具体的に言語化してください。

【私の職歴・経験】
・現在の仕事：[例：中小製造業の品質管理担当／10年]
・主な業務内容：[例：製品検査、不良品対応、社内報告書作成]
・印象に残っている仕事のエピソード：[例：不良品率を半年で30%削減した取り組みを主導した]
・周囲からよく言われること：[例：「仕事が丁寧」「報告が的確」]

強みを出したら、それぞれの強みが面接でどう活かせるかも教えてください。
```

### コピペOKプロンプト②：価値観・転職軸の整理

```
私の転職の動機と希望条件を整理して、「転職で本当に求めているもの」を明確にしてください。

【転職を考えた理由】（複数OK）
・[例：給与をもっと上げたい]
・[例：AIやDXに積極的な会社で働きたい]
・[例：残業を減らして家族との時間を増やしたい]

【転職先に求める条件】
・必須条件：[例：年収500万以上、製造業または事務職、転勤なし]
・できれば：[例：AIツールの活用が進んでいる会社、フレックスタイム]

優先順位をつけて、「この条件を絶対に妥協してはいけない」という点を教えてください。
```

### 製造業・事務職向け：「技術スキル以外」の強みをAIで引き出す

製造業や事務職の方は「特別なスキルがない」と思いがちですが、それは大きな誤解です。AIに以下のように投げかけてみてください。

```
製造業の現場担当者が持っている「IT系の人には真似できない強み」は何ですか？
以下の私の経験をもとに、転職市場で価値があると思われる強みを教えてください。
[経験の概要を入力]
```

AIが「品質へのこだわり」「段取り力」「現場課題の発見力」「報連相の正確さ」など、あなたが当たり前だと思っていた能力を「強み」として整理してくれます。

## 4. ステップ2：求人票分析——AIで「この会社に応募すべきか」を見極める

求人票は情報量が多く、「本当にいい会社かどうか」を見極めるのが難しい。AIに求人票の文章を読み込ませると、見落としがちなポイントを整理してくれます。

### 求人票のどこを読むべきか

まず自分で確認すべき4点を押さえてください。

1. **給与レンジの幅** → 幅が広すぎる（例：250〜600万）場合は注意。実態は低い可能性
2. **求める人物像の表現** → 「元気な方」「ポジティブな方」など曖昧な表現が多い場合は社風が不明確なことも
3. **業務内容の具体性** → 「その他業務」が多い場合は業務範囲が広がりやすい
4. **設立年・従業員数・離職率** → 公開されている会社は透明性が高い傾向

### コピペOKプロンプト③：求人票の読み解き

```
以下の求人票の内容を分析してください。
私は製造業・事務職で転職活動中の[年齢]歳です。

【求人票の内容】
（求人票のテキストをここに貼り付け ※会社名は「A社」に置き換えること）

以下の観点で分析してください：
1. この求人の魅力・良い点
2. 気になる点・要確認事項
3. この求人が私の転職軸（[転職軸を一文で入力]）に合っているか
4. 面接で確認すべき質問3つ
```

### コピペOKプロンプト④：求人票の「危険サイン」検出

```
以下の求人票に、ブラック企業やミスマッチを示す「危険サイン」がないか確認してください。
主に以下の観点でチェックしてください：
・給与・残業・福利厚生の表現の不自然さ
・「やりがい」「情熱」など感情的な表現への依存
・業務内容の曖昧さ
・「アットホーム」など実態が不明な表現

【求人票の内容】
（テキストをここに貼り付け）
```

## 5. ステップ3：書類作成——職務経歴書・自己PRをAIで仕上げる

書類作成は転職活動の中でも最も時間がかかる工程ですが、AIを使えば大幅に短縮できます。重要なのは、AIに「最初から完成品を作らせない」ことです。

### AIを使った書類作成の3ステップ

**ステップA: 素材を洗い出す（10分）**
ステップ1で整理した強みや経験を箇条書きでメモする。AIに渡す「材料」を先に揃える。

**ステップB: AIに下書きを作らせる（5分）**
プロンプトで素材を渡し、構成と文章の下書きを作ってもらう。

**ステップC: 自分の言葉に整える（20分）**
AIの文章を読み返し、「自分が実際に言いそうな言葉」に書き直す。AIらしい表現（「貢献してまいりました」「培ってきました」など）を自分の口語に近い表現に直すことで、面接でも同じトーンで話せる。

### コピペOKプロンプト⑤：職務経歴書の下書き作成

```
製造業・事務職向けの職務経歴書の下書きを作ってください。

【基本情報（個人名は省略）】
・職種：[例：品質管理担当]
・業種：[例：自動車部品製造業]
・在籍期間：[例：〇年〇月〜現在（〇年）]

【主な業務内容】
・[例：製品の受入・工程・出荷検査の実施（月〇〇件）]
・[例：不良品発生時の原因調査と是正処置報告書の作成]
・[例：社内品質会議での報告資料の作成]

【成果・実績（数値があれば）】
・[例：検査手順の見直しにより、見逃し件数を前年比50%削減]
・[例：月次報告資料のフォーマット改善で作成時間を2時間→30分に短縮]

フォーマット：職務概要→主要業績→スキル の順で300字程度にまとめてください。
```

### コピペOKプロンプト⑥：自己PR作成

```
以下の情報をもとに、転職面接で使える自己PR（300〜400字）を作成してください。

【私の強み（ステップ1で整理したもの）】
[例：品質への高い意識と、問題を見つけたときに根本原因まで追う粘り強さ]

【その強みを示す具体的なエピソード】
[例：〇年前、生産ラインで小さな寸法ズレが頻発する問題が起きた。原因を調べた結果、治具の摩耗が原因と突き止め、交換サイクルの変更を提案した。以来、同問題の発生はゼロになった。]

【転職先でこの強みをどう活かしたいか】
[例：品質管理の仕組みがまだ整っていない中小企業で、現場目線の改善活動を推進したい]

自然な話し言葉に近い文体で書いてください。
```

## 6. ステップ4：面接準備——AIで想定Q&Aを作り込む

面接で緊張する最大の原因は「何を聞かれるかわからない」という不安です。AIで想定Q&Aを事前に作り込んでおけば、本番の心理的余裕が大きく変わります。

### 製造業・事務職でよく聞かれる質問6選

1. 「転職理由を教えてください」
2. 「前職での一番の実績・成果は何ですか？」
3. 「弊社を選んだ理由を教えてください」
4. 「入社後、どのように貢献できると思いますか？」
5. 「あなたの弱みを教えてください」
6. 「5年後、どのようなキャリアを目指していますか？」

### コピペOKプロンプト⑦：想定Q&A作成

```
製造業・事務職の中途転職の面接で想定される質問と、私の状況に合わせた回答案を作ってください。

【私の基本情報】
・転職理由（ポジティブな表現で）：[例：より品質改善に専念できる環境を求めて]
・強み：[例：品質管理の実務経験10年、報告書作成が速い]
・希望：[例：品質管理・製造管理職]

以下の質問について、各200字程度の回答案を作成してください。
1. 転職理由
2. 前職での最大の実績
3. 志望動機
4. 入社後の貢献
5. 自分の弱み（改善中であることも含めて）
```

### コピペOKプロンプト⑧：模擬面接（壁打ち）

```
あなたは採用担当者です。私に対して製造業の品質管理職の面接を行ってください。
質問は1問ずつ出してください。私が回答したら、次の質問に進む前に
「回答の良い点」と「もっとこうするとよい点」をフィードバックしてください。
では、始めてください。
```

このプロンプトで「1問ずつ回答→フィードバック→次の質問」を繰り返すことで、本番さながらの練習ができます。移動中・昼休みなど、隙間時間に使えるのが最大のメリットです。

```

・ポジション：[職種名]
・選考状況：書類審査中 / 1次面接 / 最終面接 / 内定 / 不採用
・次のアクション：[例：〇月〇日 1次面接]
```
