# ChatGPTで研修・OJT資料を爆速作成する方法【製造業・管理職向けコピペOKプロンプト付き2026年版】

「新人が入ってきたのに、教えるための資料がない」——製造業の現場でよく聞く話です。手順書を一から作ろうとすると、それだけで丸1日つぶれてしまうこともあります。この記事では、**ChatGPTを使って研修・OJT資料を短時間で作る方法**を、コピペOKのプロンプトとともに解説します。

---

## 製造業のOJTは「資料作成の負担」が大きい

厚生労働省の令和6年度「能力開発基本調査」によると、計画的なOJTを実施している事業所は**61.1%**にのぼります。新入社員への実施が最も多く（54.7%）、毎年4〜5月になると多くの管理職・班長が「教育担当」の仕事を抱えます。

問題は、OJT用の資料を整備する時間がなかなか確保できないことです。

- 「前任者から引き継いだ手順書が古いまま」
- 「口頭で教えるだけで、チェックリストが存在しない」
- 「毎回ゼロから説明しているので、教える人によって内容がバラバラ」

ChatGPTを使うと、こうした資料作成の「構成を考える時間」「文章を書く時間」を大幅に短縮できます。

---

## ChatGPTでOJT資料を作る3つのメリット

### 1. 「何を教えればいいか」を整理してくれる

経験豊富な管理職ほど「当たり前すぎて文章にできない」という壁にぶつかります。ChatGPTに「この作業のOJT資料を作って」と伝えると、ベテランが見落としがちな基礎的なステップまで網羅した構成を提案してくれます。

### 2. 読み手のレベルに合わせた文章を生成できる

「新入社員向け」「製造経験あり・工場は未経験の中途向け」など、対象者を指定するだけで文体・説明の深さを調整してくれます。

### 3. チェックリスト・評価シートも同時に作れる

手順書だけでなく、「習得確認チェックリスト」「振り返りシート」もまとめて生成できます。OJTの記録として残せる形まで一気に整います。

## 方法①：業務手順説明スライドを作る

新人に業務の流れを説明するスライドや資料の骨子を作ります。

### 手順

1. 以下のプロンプトをChatGPTに貼り付ける
2. 出力された内容を確認・修正する
3. PowerPointまたはWordに貼り付けて体裁を整える

### コピペOKプロンプト

**【製造ライン・作業手順説明向け】**
```
あなたは製造現場の熟練リーダーです。
新入社員（製造業は初めて）向けのOJT資料を作成してください。

【対象業務】：射出成形機のオペレーター業務
【対象者】：製造業未経験の新入社員
【ゴール】：1人で安全に基本オペレーションができるようになること
【研修期間の目安】：2週間

以下の形式で出力してください：
1. この業務を覚えるうえで最初に知るべきこと（安全・全体像）
2. 1日目〜3日目に習得すべき基本手順（ステップ形式）
3. 4日目〜1週間で習得すべき応用手順
4. 2週間後の習得確認チェックリスト（10項目）
5. よくあるミスと対処法（3〜5件）
```

**【事務・間接部門向け】**
```
あなたは事務部門のベテランスタッフです。
中途入社者（事務経験あり・この会社が初めて）向けのOJT資料を作成してください。

【対象業務】：受発注管理業務（基幹システムへのデータ入力・在庫確認）
【対象者】：他社での事務経験3年以上・当社システムは未経験
【ゴール】：1人で日常の受発注処理を完結できるようになること
【研修期間の目安】：1週間

以下の形式で出力してください：
1. 業務全体の流れ（図解のテキスト版）
2. 1日目〜3日目の習得ステップ
3. よく迷うポイントとその対処法（5件）
4. 習得確認チェックリスト（8〜10項目）
```

## 方法②：評価シート・フィードバックコメントを作る

OJTの進捗を確認する「評価シート」と、指導者が書く「フィードバックコメント」もChatGPTで作れます。

### コピペOKプロンプト

**【OJT評価シート作成向け】**
```
製造現場の新入社員向けOJT評価シートを作成してください。

【評価対象業務】：フォークリフトを使った入出荷作業
【評価時期】：入社1ヶ月後
【評価者】：直属の班長

以下の形式で出力してください：
- 評価項目（10〜12項目）
  各項目に：「できている／おおむねできている／要指導」の3段階評価欄
- 総合コメント記入欄（自由記述）
- 次のステップ（1ヶ月後の目標）記入欄
```

**【フィードバックコメント文案向け】**
```
以下の状況に合わせたOJTフィードバックコメントを3パターン作成してください。

【状況】：入社2週間の新入社員（やる気はあるが、作業スピードが遅い）
【コメントの目的】：本人のモチベーションを維持しながら改善点を伝える
【文体】：丁寧かつ具体的（上司が部下に伝える形式）

各パターンに：
- 良かった点（1〜2文）
- 改善点（1〜2文）
- 次の目標（1文）
を含めてください。
```

## 方法③：月間OJTカリキュラムを作る

「何を・いつまでに・どのレベルまで」を整理した月間スケジュールもChatGPTで作れます。

### コピペOKプロンプト

**【新入社員1ヶ月カリキュラム向け】**
```
製造業の新入社員向け、入社後1ヶ月間のOJTカリキュラムを作成してください。

【職場】：金属加工工場（プレス・溶接・塗装ラインあり）
【対象】：製造業未経験の新卒社員
【1ヶ月後のゴール】：基本的な安全ルールと1つのラインで独立作業ができること

以下の形式で出力してください：
- 第1週：○○を重点的に学ぶ（主な内容3〜5項目）
- 第2週：○○を重点的に学ぶ（主な内容3〜5項目）
- 第3週：○○を重点的に学ぶ（主な内容3〜5項目）
- 第4週：習得確認・フォロー面談（内容詳細）
- 各週の「到達基準」（評価の目安）
```

---

## プロンプトを使うときの3つのコツ

### コツ1：「対象者」「ゴール」「期間」を最初に明示する

この3つを入れるだけで、出力内容の精度が大きく上がります。「新入社員向け」とだけ書くより、「製造業未経験・高卒・入社4月」のように具体的にするほど現場感のある資料になります。

### コツ2：自社特有の用語はそのまま入れてOK

「5S」「KY（危険予知）」「ヒヤリハット報告」「QC活動」など、製造業の専門用語はそのまま書いて構いません。ChatGPTは業界用語を理解したうえで資料を作ってくれます。

### コツ3：出力が長すぎるときは「要約して」と追加指示

チェックリストが20項目になってしまった場合は「10項目以内に絞って」と追加指示するだけで調整できます。

## よくある質問

**Q：作った資料はそのまま使えますか？**
ChatGPTが出力した内容は「たたき台」として使うのが最適です。自社の機械・工程・社内ルールに合わせて必ず確認・修正してから使用してください。

**Q：無料版のChatGPTでも使えますか？**
方法①〜③はすべて無料版で使えます。ChatGPT Plusでは出力が速く、一度により多くの内容を生成できます。

**Q：作った資料の著作権は誰のものですか？**
ChatGPTの出力物の著作権はユーザーに帰属するとされています（OpenAIの利用規約に基づく）。ただし、社内利用の際は自社のAI利用ポリシーも確認してください。

**Q：機密情報を入力しても大丈夫ですか？**
会社固有の機密情報（顧客情報・設計データなど）は入力しないことを強くお勧めします。プロンプトには「フォークリフト作業」「受発注業務」といった一般的な業務内容の記述に留め、固有名詞・数値データは社内で追記する形にしてください。

## まとめ：OJT資料の「骨格」はChatGPTに作らせる

ChatGPTでOJT・研修資料を作る方法を3つ紹介しました。

| 方法 | 向いている場面 | 出力物 |
|-----|------------|-------|
| ①業務手順説明 | 作業手順書・説明資料がない | スライド骨子・手順書 |
| ②評価・フィードバック | 評価シートや指導コメントに困っている | 評価シート・コメント文案 |
| ③月間カリキュラム | OJTの全体設計をしたい | 月間スケジュール・到達基準 |

厚生労働省の調査では、6割超の事業所がOJTを実施しています。しかし「資料が整っていない」「毎回口頭で教えている」という現場はまだ多いのが実態です。ChatGPTを使えば、骨格となる資料を短時間で作り、その分を「実際に教える時間」「フォローする時間」に充てられます。

まずは方法①のプロンプトを1つコピペして、自分の職場に当てはめてみてください。
