## 2. 使う前の注意点：機密情報のマスキング（1分で確認）

改善提案書にはコスト数値・設備名・仕入先名など、社外に出してはいけない情報が含まれることがあります。AIに入力する際は以下のように置き換えてください。

| 入力してはいけない情報 | 置き換え方 |
|------------------|---------|
| 仕入先・取引先の社名 | 「A社」「仕入先B」 |
| 社内の設備名・型番 | 「設備X」「〇〇ライン」 |
| 具体的なコスト金額 | 「月XX万円程度」（桁のみ） |
| 顧客名 | 「お客様」「C社様」 |

## 3. 改善提案書をChatGPTで作る方法

### 改善提案書の必要要素

改善提案書は「問題→原因→改善策→効果→スケジュール」の5要素で構成されます。

- **現状の問題：** 何が、どこで、どの程度起きているか
- **問題の原因：** なぜ起きているか（推定でよい）
- **改善提案の内容：** 具体的に何をどう変えるか
- **期待される効果：** 改善後にどうなるか（定量・定性）
- **実施スケジュール：** いつ誰が対応するか

この5要素をChatGPTに渡すだけで、読み手に伝わる提案書の下書きが完成します。

### コピペOKプロンプト①：日常カイゼン提案書

```
製造業向けの改善提案書の下書きを作成してください。

【現状の問題】
・問題の内容：[例：溶接工程で作業者が長時間前かがみの姿勢を取っており、腰痛を訴える件数が増えている]
・発生場所：[例：第2製造ライン 溶接工程]
・発生頻度・規模：[例：月に2〜3名が腰痛を訴え、うち1名は先月3日間休業した]

【問題の原因（推定）】
・[例：作業台の高さが作業者の身長に合っておらず、標準姿勢を維持しにくい]
・[例：作業台高さ調整の仕組みがなく、全員同じ高さで作業している]

【改善提案の内容】
・[例：高さ調整機能付きの作業台を導入し、作業者ごとに適切な高さに設定できるようにする]
・[例：導入前に全作業者の適正作業台高さを計測・記録する]

【期待される効果】
・[例：腰痛による休業件数の削減]
・[例：作業者の疲労軽減による品質安定]

【実施スケジュール（目安）】
・[例：来月中に作業台を1台試験導入し、3ヶ月間効果を検証する]

上記を「1. 現状と課題 / 2. 原因分析 / 3. 改善提案内容 / 4. 期待効果 / 5. 実施計画」の
構成で、社内提案書として読みやすくまとめてください。
提案者が現場担当者であることを踏まえ、わかりやすい言葉で作成してください。
```

### コピペOKプロンプト②：コスト削減提案書

```
製造業向けのコスト削減提案書の下書きを作成してください。

【削減対象のコスト】
・対象：[例：包装資材（緩衝材）の使用量]
・現在のコスト概算：[例：月に約XX万円分の緩衝材を使用している]
・問題点：[例：過剰梱包になっており、同じ保護性能をより少ない材料で実現できる余地がある]

【改善提案の内容】
・[例：緩衝材の厚みと枚数を見直し、最低限必要な仕様に標準化する]
・[例：社内で検証試験を行い、品質保証部門の承認を得たうえで変更する]

【期待される削減効果（試算）】
・[例：現状比で緩衝材使用量を2〜3割削減できると見込んでいる]
・[例：年間でXX万円前後の削減になる見込み（正確な試算は検証後に算出）]

【リスクと対策】
・[例：梱包品質の低下リスク → 試験ロットで輸送テストを行い品質確認後に本格展開]

上記を「1. 削減対象と現状コスト / 2. 改善提案内容 / 3. 期待される削減効果 / 4. リスクと対策 / 5. 推進スケジュール」の構成でまとめてください。
経営陣・上司への提出を想定した、数値根拠を重視したビジネス文書として作成してください。
```

## 4. 5S活動報告書をChatGPTで仕上げる方法

5S（整理・整頓・清掃・清潔・躾）の活動報告書は、毎月定例で作成する書類です。「活動はしたが、成果が文章で伝わらない」という悩みを持つ担当者が多い書類でもあります。

### 5S活動報告書の構造

```
1. 活動テーマ（5Sのどの項目に取り組んだか）
2. 活動内容（いつ、誰が、何をしたか）
3. 改善前の状態（Before）
4. 改善後の状態（After）
5. 効果・成果（定量・定性の両面）
6. 次月の活動計画
```

### コピペOKプロンプト③：5S活動報告書

```
製造業の5S活動報告書の下書きを作成してください。

【活動の概要】
・実施月：[例：2026年4月]
・担当エリア：[例：第1倉庫 資材置き場]
・取り組んだ5S項目：[例：整理・整頓]

【活動内容】
・[例：不要な資材・廃材を仕分けし、使用頻度の低いものをエリア外に移動した]
・[例：よく使う資材の置き場所を決め、床にテープでエリア表示を行った]
・[例：活動参加者：Aライン全員（8名）、活動時間：月2回×各2時間]

【改善前の状態（Before）】
・[例：資材の置き場所が決まっておらず、必要な資材を探すのに毎回時間がかかっていた]
・[例：通路に資材がはみ出しており、フォークリフトの通行に支障が出ていた]

【改善後の状態（After）】
・[例：資材ごとに置き場所を固定し、どこに何があるかひと目でわかるようになった]
・[例：通路を確保し、フォークリフトが安全に通行できるようになった]

【効果・成果】
・[例：資材を探す時間が短縮された（目安：1回あたり5分程度の削減）]
・[例：通路の安全性が向上し、ヒヤリハットゼロを継続中]

【来月の活動計画】
・[例：清掃の標準化に取り組む。エリアごとの清掃担当と手順を決め、清掃チェックシートを作成する]

上記を「1. 活動テーマ / 2. 活動内容 / 3. 改善前後の比較（Before→After） / 4. 効果・成果 / 5. 来月の計画」の構成でまとめてください。
上司・管理職への報告を想定した、成果が伝わりやすい文体でお願いします。
```

### 「成果が伝わらない」よくある失敗とAIで直す方法

5S報告書でよく見られる「成果が伝わらない」失敗例と、AIでの改善方法を示します。

**❌ 成果が見えない書き方の例:**
> 「整理整頓を実施しました。職場がきれいになりました。」

**✅ 成果が伝わる書き方の例:**
> 「使用頻度の低い資材47点を別エリアへ移動し、作業エリアを約2割広く確保しました。資材の探し出し時間が短縮され、作業開始のスムーズさが向上しました。」

AIに「この成果の記述は抽象的すぎないか、具体的な変化として書き直してほしい」と依頼することで、審査・評価で高評価を得やすいレベルまで引き上げられます。

```
以下の5S活動報告書の「成果・効果」の記述を改善してください。
「何がどう変わったか」が具体的に伝わる表現に書き直してください。

現在の記述：[例：整理整頓を実施し、職場環境が改善されました]
活動の実態：[例：資材の置き場所を決めてエリア表示した。探し時間が減った]
```

## 5. 改善活動計画書をChatGPTで作る方法

### コピペOKプロンプト④：改善活動計画書

```
製造業の改善活動計画書の下書きを作成してください。

【活動の目的】
・[例：〇〇工程における段取り替え時間の短縮]
・[例：現状は段取り替えに平均45分かかっており、30分以内を目標とする]

【活動メンバー】
・[例：リーダー：山田（製造2課）、メンバー：鈴木・田中・佐藤（各1名）]

【活動期間】
・[例：2026年5月〜2026年7月（3ヶ月間）]

【主な活動ステップ（案）】
・[例：5月：現状の段取り作業を動画で記録し、時間分析を行う]
・[例：6月：時間のかかっている作業を特定し、改善案を検討・試行する]
・[例：7月：改善後の効果を検証し、標準化・横展開を実施する]

【目標（KPI）】
・[例：段取り替え時間：45分 → 30分以内（33%短縮）]

上記を「1. 活動目的 / 2. 体制・メンバー / 3. 活動スケジュール / 4. 目標（KPI） / 5. 進捗確認の方法」の構成でまとめてください。
キックオフ会議での配布資料として使えるレベルの書き方でお願いします。
```

## 6. まとめ：「書く時間」を「考える時間」に変える3ステップ

### 着手の優先順位

| 優先度 | 書類の種類 | 理由 |
|-------|----------|------|
| 1位 | 日常カイゼン提案書 | 発生頻度が最も高く、積み重ねの効果が大きい |
| 2位 | 5S活動報告書 | 月1回の定例業務を完全ルーティン化できる |
| 3位 | コスト削減提案書 | 経営インパクトが大きく、AIで根拠付けが強化できる |
| 4位 | 改善活動計画書 | 作成頻度は低いが時間がかかるため効果大 |

### 今週の3アクション

| # | アクション | 目安時間 |
|---|-----------|---------|
| 1 | 直近で提出した（または提出できなかった）改善提案書を1件思い出し、プロンプト①の空欄を埋めてみる | 10分 |
| 2 | ChatGPTで下書きを生成し、実際の提案書と品質・時間を比較する | 20分 |
| 3 | 「AIで下書き、自分は確認・修正だけ」というルールをチームで合意する | 朝礼5分 |
