## 2. Perplexity AIとChatGPTの決定的な3つの違い

Perplexityを正しく使うためには、ChatGPTとの違いを理解することが出発点になります。

### 違い①：リアルタイム検索 vs 学習データ依存

ChatGPTは学習データのカットオフ（情報の締め切り日）が存在します。2026年4月時点では、数ヶ月〜1年以上前の情報が「最新情報」として返ってくる場合があります。

Perplexityはこの問題を解決するため、**質問のたびにWebをリアルタイムで検索**して回答を生成します。今日の朝に発表されたニュースや、先月改訂された規格情報にも対応できます。

### 違い②：出典付き回答 vs 出典なし

ChatGPTはデフォルトでは回答に出典を示しません。そのため「本当に正しいのか」を確認するために、別途Googleで調べ直す手間が発生します。

Perplexityは**すべての回答に情報ソースのURLを付けて表示**します。「この情報の根拠はどこか」をすぐに確認でき、社内説明資料や上司への報告で使う際の信頼性が大幅に上がります。

### 違い③：「検索に近い」vs「対話・文書作成に強い」

Perplexityは「最新の正確な情報を素早く調べる」ことに特化しています。一方、ChatGPTは長文の文書作成・プロンプトに沿った文章生成・複雑な思考の壁打ちに強みがあります。

### どちらを使うべきか——2択の判断フロー

```
今やろうとしていることは？

「最新情報を調べたい」「出典付きで情報を確認したい」
       → Perplexity AIを使う

「文書を作りたい」「アイデアを出したい」「長い文章を整えたい」
       → ChatGPTを使う

どちらか迷ったら → Perplexityで情報収集してからChatGPTで文書化
```

## 4. 仕事で使える8つのシーン（製造業・事務職特化）

### シーン①：規制・法改正の最新情報を確認する

製造業の現場では、労働安全衛生法の改正・化学物質規制の変更・環境規制の強化など、法改正への対応が常に求められます。Perplexityなら「今この瞬間の最新情報」を官公庁・業界団体のサイトを出典に回答してくれます。

**コピペOKプロンプト①**
```
2026年に施行または改正予定の製造業に関係する法律・規制を教えてください。
特に以下の分野を重点的に調べてください：
・労働安全衛生法
・化学物質管理規制（GHS・SDS関連）
・環境規制（廃棄物・排水）

各項目について「いつから」「何が変わるか」「中小企業が対応すべきこと」を簡潔にまとめてください。
出典URLも一緒に示してください。
```

---

### シーン②：材料・資材の価格動向を調べる

調達担当や原価管理担当にとって、鋼材・アルミ・樹脂・銅などの材料価格動向は毎月チェックが必要な情報です。Perplexityなら最新の業界ニュースや統計データを集約して回答してくれます。

**コピペOKプロンプト②**
```
2026年4月時点の以下の材料・資材の価格動向を教えてください。
・熱延鋼板
・アルミニウム地金
・電気銅

各材料について「現在の相場水準」「前月・前年同期比」「今後の見通し」を
出典付きで教えてください。情報が不確かな部分は「要確認」と記載してください。
```

---

### シーン③：競合他社・取引先の最新動向を収集する

新製品発売・M&A・工場新設・業績など、競合他社や取引先の動向を定期的に把握しておくことはビジネスの基本です。Perplexityで検索すると、直近のプレスリリースやニュース記事をまとめて回答してくれます。

**コピペOKプロンプト③**
> ⚠️ 競合・取引先の社名は調査目的のため入力OKですが、社内機密情報（契約内容・仕入れ価格等）は含めないこと。

```
[会社名または業界名]に関する直近3ヶ月の主なニュース・動向を教えてください。
以下の観点でまとめてください：
・新製品・新サービスの発表
・経営・人事・組織変更
・業績・財務関連の発表
・業界全体のトレンド

各情報の出典URLを添付してください。
```
