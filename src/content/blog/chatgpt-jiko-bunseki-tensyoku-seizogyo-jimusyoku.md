ChatGPTは、この「どうやって整理すればいいかわからない」状態を動かすのに役立ちます。

## なぜ自己分析に「壁打ち相手」が必要なのか

自己分析が進まない最大の理由は、「自分一人で考えると堂々巡りになる」ことです。

「今の仕事が嫌なのはわかっている。でもなぜ嫌なのか、どんな仕事なら向いているのかがわからない」
——この状態で自己分析のワークシートを広げても、なかなか前に進みません。

人は「聞いてくれる相手がいる」ことで、自分の考えを言語化しやすくなります。
しかし、転職を考えていることを友人や家族に話すのは気が引けることも多いです。

ChatGPTは、この「壁打ち相手」として機能します。

- 否定や批判をしない
- 「もう少し詳しく教えてください」と掘り下げてくれる
- 時間・場所を選ばず、何時間でも付き合ってくれる

自分の考えを整理するための最初の一歩として、ChatGPTを活用することができます。

> **注意:** ChatGPTはあくまで「考えを整理する道具」です。転職の最終的な判断や、具体的な求人の紹介・選考対策については、転職エージェントなど専門家への相談を組み合わせることをおすすめします。

## ChatGPTで自己分析を進める3つのステップ

### ステップ①：今の仕事を「棚卸し」する

まず、今の仕事で「良かったこと」と「嫌だったこと」を洗い出します。
ここで大事なのは、評価や判断をせず、思ったことをそのまま出すことです。

**コピペOKプロンプト：仕事の棚卸し**

```
あなたはキャリアコンサルタントです。
私が今の仕事について話すので、「良かった点」「嫌だった点」「印象に残っていること」に分類して整理してください。
また、整理した内容から、私が仕事に何を求めているかを読み取れる点があれば指摘してください。

【私の仕事の状況】
・職種・業種：（例：製造業の品質管理、5年目）
・良かったこと：（例：工程改善の提案が採用されたとき、達成感があった）
・嫌だったこと・つらかったこと：（例：上司への報告のための書類作成が毎週あり、本来の仕事に集中できない）
・印象に残っている出来事：（例：自分が提案した手順書が現場で使われ続けているのを見た）

【出力形式】
・「良かった点」「嫌だった点」「印象に残っていること」に分類して箇条書きで整理
・整理した内容から読み取れる「私が仕事に求めているもの」を3点以内で提示
```

**使い方のポイント：**
完璧な文章でなくていいです。「嫌だったこと」欄には、愚痴でも構いません。
「こんなこと言っていいのかな」と感じることほど、自己分析の核心に近いことが多いです。

---

### ステップ②：「強み」と「苦手なこと」を整理する

次に、今までの仕事経験から「自分が得意なこと・苦手なこと」を掘り出します。
自分では「当たり前」に感じていることが、実は他の人にはできない強みである場合があります。

**コピペOKプロンプト：強みと苦手の整理**

```
あなたはキャリアコンサルタントです。
以下の私の仕事経験から、「強み（得意なこと）」と「苦手なこと」を整理してください。
また、「強み」の中で、転職市場で評価されやすいものがあれば教えてください。

【私の仕事経験】
・これまでやってきた業務：（箇条書きで記入）
・「これは得意だ」と感じた場面：（例：複雑な手順書をわかりやすくまとめることが得意）
・「これは苦手だ・やりたくない」と感じた場面：（例：大人数の会議で発言するのが得意でない）
・周囲から「助かった」「ありがとう」と言われた経験：（例：議事録や報告書の作成を頼まれることが多い）

【出力形式】
・「強み」3〜5点を箇条書きで整理
・「苦手なこと」2〜3点を箇条書きで整理
・強みの中で転職市場での評価が高そうなものに「★」をつけて理由を一言添える
```

**使い方のポイント：**
「周囲から助かったと言われた経験」が特に重要です。
自分では「大したことない」と思っていても、他者から見ると希少なスキルであることが多いです。

---

### ステップ③：「転職の軸」を言語化する

自己分析の最終ゴールは「次の仕事で何を大切にするか（転職の軸）」を言語化することです。
転職の軸がないと、求人を見ても「どれも同じに見える」「迷ってばかり」という状態になります。

**コピペOKプロンプト：転職の軸の言語化**

```
あなたはキャリアコンサルタントです。
以下の私の状況から、「転職の軸（次の仕事で何を優先するか）」を整理してください。

【私の状況】
・現在の不満・改善したいこと：（例：残業が多く、家族との時間が取れない）
・次の仕事に求めること：（例：成果が数字で見えやすい仕事がしたい。給与よりも裁量の大きさを重視したい）
・絶対に嫌なこと・譲れない条件：（例：転勤はNG、夜勤はNG）
・理想の5年後のイメージ（あれば）：（例：専門スキルを活かして独立できる状態になりたい）

【出力形式】
・「転職の軸」を優先順位をつけて3〜5点で整理
・それぞれの軸について「なぜそれを重視するのか」を1行で補足
・この軸に合いそうな職種・業種のイメージを3つ程度提示（参考として）
```

**使い方のポイント：**
「転職の軸」が出てきたら、求人を眺める際にその軸に照らし合わせてみましょう。
「この求人は軸①と②は合うけど③が微妙」という基準で見られるようになります。

## 製造業・事務職のための「キャリア相談プロンプト」

### 「AIに代替されにくいスキル」を自分の仕事から探す

AI活用が広がる中で、「自分の仕事はAIに取られるのでは」という不安を抱えている方も多いです。
ChatGPTに自分の仕事内容を伝えて、代替されにくい要素を整理してもらうことができます。

```
あなたはキャリアコンサルタント兼AI活用の専門家です。
以下の私の仕事内容について、「AIに代替されにくい要素」と「AIが強化してくれる要素」を整理してください。
また、AI活用スキルを加えることでどんな価値が生まれるか、具体的な例を教えてください。

【私の仕事内容】
（今やっている主な業務を箇条書きで記入）
```

---

### 「転職か副業か、どちらが今の自分に向いているか」を整理する

「転職が正解なのか、まず副業でスキルを試すべきか」という判断は難しいです。
ChatGPTに状況を伝えて、選択肢のメリット・デメリットを整理してもらうことができます。

```
あなたはキャリアコンサルタントです。
私の状況を踏まえて、「転職」と「副業スタート」それぞれのメリット・デメリットを整理してください。
どちらが今の私に向いているかについて、あなたの見解も聞かせてください（断定ではなく、考え方の参考として）。

【私の状況】
・年齢：（例：35歳）
・家族構成：（例：配偶者あり、子供1人）
・現在の年収・生活費の状況：（例：年収400万円。生活費を考えると収入が下がるのは1年以上は厳しい）
・転職を考えている理由：（例：現在の業界の将来性に不安。AIスキルを活かせる仕事をしたい）
・副業でやれそうなこと：（例：ChatGPTを使った文書作成代行なら始められそう）
```

## ChatGPT自己分析の「限界」と転職エージェントとの使い分け

ChatGPTを使った自己分析には、有効な面と限界の両方があります。

### ChatGPTが得意なこと

- 自分の考えを「整理・言語化」すること
- 思考が堂々巡りになっているときに「別の視点」を提示すること
- 何時間でも付き合いながら、考えを深掘りすること
- 「転職の軸」の草案を作ること

### ChatGPTが苦手なこと

- **現在の転職市場の情報:** ChatGPTは最新の求人動向や業界の採用状況を知りません。「あなたのスキルは今の市場でいくらの価値があるか」という客観的な評価は返せません。
- **あなた個人への具体的なアドバイス:** ChatGPTが返す内容はあなたが入力した情報だけに基づいています。面接での強みの伝え方など、個人に合った具体的な指導は転職エージェントの方が適しています。
- **求人の紹介・マッチング:** ChatGPTは実際の求人を持っていません。

### 効果的な使い分け

| 用途 | ChatGPT | 転職エージェント |
|------|---------|----------------|
| 自己分析・考えの整理 | ◎ | △（時間がかかる） |
| 転職市場の相場・動向確認 | △ | ◎ |
| 職務経歴書・履歴書のたたき台作成 | ◎ | ○ |
| 面接対策（想定Q&Aの生成） | ◎ | ◎ |
| 求人の紹介・マッチング | ✕ | ◎ |

**推奨フロー:**
1. ChatGPTで自己分析・転職の軸を整理（本記事の手順）
2. 整理した内容を転職エージェントに持ち込んで、市場での価値を確認
3. エージェントのサポートを受けながら求人探しと選考を進める

> ▶ [doda（無料転職相談・求人紹介）](https://doda.jp/)
>

> ▶ [リクルートエージェント（無料登録）](https://www.r-agent.com/)

## まとめ——ChatGPTは「自分の考えを整理する壁打ち相手」

転職活動で最初につまずく「自己分析」は、一人で考えると堂々巡りになりがちです。
ChatGPTを壁打ち相手にすることで、「自分が何をしたいのか」「何が得意なのか」を言語化しやすくなります。

**今日からできる3つのアクション:**
1. このページの「仕事の棚卸しプロンプト」を使って、今の仕事を整理してみる
2. 「転職の軸プロンプト」で、次の仕事に求めることを3点以上書き出す
3. 軸が整理できたら、転職エージェントへの相談を検討してみる

転職は大きな決断です。
ChatGPTで考えを整理し、専門家の視点を組み合わせることで、後悔の少ない選択につなげてください。
