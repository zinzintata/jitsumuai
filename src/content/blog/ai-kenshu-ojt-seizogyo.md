## 結論：ChatGPTで研修資料の作成時間を大幅に削減できる可能性がある

4月は新人・中途社員の受け入れが集中し、教育担当者の負荷が一気に高まる時期です。
OJT計画書・作業手順書・安全教育テキストなど、複数の教材を本来業務と並行して作らなければならない状況に、「毎年同じことの繰り返しなのに終わらない」と感じている方も多いのではないでしょうか。

**この記事の結論を先にお伝えします。**
ChatGPT（チャットGPT）などの生成AIを活用すると、研修資料の「たたき台」を短時間で作成できる可能性があります。
すべてをAIに任せることはできません。
ただし、構成案・本文・チェックリスト・クイズ問題などの下書き生成をAIに担わせることで、教育担当者は「確認・加筆・品質担保」に集中できるようになります。

### 4月に追われる教育担当者が今すぐできること

**今すぐ取り組めることは、「入社1日目の案内資料1枚」をAIで下書きさせることです。**
複雑な安全教育テキストや作業手順書から始める必要はありません。
まずスコープを最小化することで、AIの使い心地をつかみながら確実に成果を出せます。

具体的には、ChatGPT・Microsoft Copilot（コパイロット）・Google Gemini（ジェミニ）のいずれかを使い、「新入社員向けに〇〇を案内する1枚ペーパーを作ってください」と指示するだけで構成案が得られます。
小さな成功体験を積み重ねることが、AI活用を現場に定着させる最初の一歩です。

### AIで作れる教材の種類と「人間が必ず加える部分」の切り分け

**AIが得意な作業と、人間が担うべき作業を最初に区別しておくことが重要です。**
AIを万能ツールとして過信すると、事実誤認や機密漏洩のリスクにつながります。

AIが得意なのは「構成を考える」「文章の下書きを作る」「表現をやさしく言い換える」作業です。
一方、「数値・法令基準の正確な記述」「社内固有のルールの反映」「現場感覚に基づいた加筆」は、人間が必ず担う必要があります。

| AI担当（下書き） | 人間担当（確認・加筆） |
|----------------|----------------------|
| 目次・章立ての提案 | 社内規定との整合確認 |
| 本文のたたき台生成 | 数値・法令基準の事実確認 |
| チェックリスト・クイズの作成 | 現場経験者によるレビュー |
| やさしい日本語への変換 | 最終版の承認・署名 |

この役割分担を守ることで、AI活用のメリットを安全に享受できます。

### この記事で再現できること・できないことの整理

**この記事では「たたき台作成の時間短縮」は再現できますが、「教材の品質保証」はできません。**
AIが生成したテキストには誤情報が含まれる可能性があります。
特に安全教育や法令関連の内容は、現場責任者・安全管理者によるレビューが必須です。

再現できること：OJT計画書・作業手順・チェックリスト・クイズ問題の下書き生成、やさしい日本語への変換
再現できないこと：社内固有ルールの自動反映、数値・法令基準の自動検証、現場レビューの代替

この記事を読むことで、「どのプロンプト（AIへの指示文）を使えばよいか」「どこまでAIに任せてよいか」が整理できます。

## Before/After比較：手作業 vs AI活用の研修資料作成フロー

**AI活用によって、研修資料作成の「たたき台を作る時間」を大幅に短縮できる可能性があります。**
ここでは手作業の現実と、AI活用後のフローを比較して、変化のイメージを具体的につかみましょう。

### Before：本来業務の合間に資料を一から作る現実

**手作業での資料作成は、構成を考えるところから始まるため時間がかかりがちです。**
過去の資料を探し、フォーマットを整え、文章を一から書き、印刷して配布する一連の流れは、本来業務と並行するには負担が大きい作業です。

たとえば「製造ラインの新人向け作業手順チェックリスト」を1枚作るだけでも、構成検討・文章作成・体裁整備で2〜4時間程度かかることがあります（業種・内容によって異なります）。
さらに4月は複数の教材を同時に準備する必要があるため、残業や休日対応が発生しやすい時期でもあります。

教育担当者の本来の役割は「教えること」であり、「資料を作ること」に全ての時間を使うのは本末転倒といえるかもしれません。

### After：AIでたたき台を作り、確認・加筆に集中する

**AI活用後のフローでは、構成案と本文のたたき台をAIが生成し、人間は確認・加筆・承認に専念できます。**
たたき台の生成にかかる時間を短縮できる分、現場レビューや品質確認に時間を使えるようになります。

| 作業フェーズ | Before（手作業） | After（AI活用） |
|------------|----------------|----------------|
| 構成案の作成 | 30〜60分 | 5〜10分（AIに指示→修正） |
| 本文のたたき台 | 60〜120分 | 10〜20分（AI生成→確認） |
| チェックリスト作成 | 30〜60分 | 5〜10分（AI生成→加筆） |
| 現場レビュー・修正 | 30〜60分 | 30〜60分（変わらない） |
| **合計（目安）** | **約3〜5時間** | **約1〜2時間** |

※上記はあくまで目安であり、教材の種類・規模・社内確認フローによって大きく異なります。

この時間差の分だけ、教える側が「実際の指導」により多くの時間を割けるようになります。

### 効果が出やすいコンテンツ類型・出にくいケース

**AI活用の効果が出やすいのは「構造が決まっているコンテンツ」です。**
チェックリスト・クイズ問題・業務フロー説明テキストなど、一定のフォーマットに当てはめる作業はAIが得意とするところです。

一方、効果が出にくいケースとして以下が挙げられます。

- 社内独自の用語・略称が多い内容（AIが文脈を知らないため誤りが増えやすい）
- 数値・法令・安全基準が核心となる内容（必ず専門家・担当者がレビューする必要がある）
- 「現場の雰囲気・暗黙知」を伝えるOJT内容（経験者の言葉でしか伝わらない部分がある）

効果が出やすい領域から着手し、慣れたら応用範囲を広げていくことをお勧めします。

## 準備：AIで研修資料を作り始める前に決めておくこと

**AIでの資料作成を始める前に、「使うツール」「教材の型」「情報取り扱いルール」の3点を決めておくと、作業がスムーズに進みます。**
準備なしに始めると、後から方針変更が必要になり、かえって手間がかかる場合があります。

### 使うAIツールの選択（ChatGPT / Copilot / Gemini）

**教育担当者が研修資料作成に使いやすいAIツールとして、ChatGPT・Copilot・Geminiの3種が代表的です。**
それぞれ特性が異なるため、職場の環境に合わせて選ぶことが重要です。

| ツール | 特徴 | 向いている用途 |
|--------|------|--------------|
| ChatGPT（OpenAI社） | 文章生成が得意・操作が直感的 | 本文生成・クイズ作成・言い換え |
| Microsoft Copilot | Word・Excelと連携しやすい | 既存ドキュメントの加工・整形 |
| Google Gemini | Google Workspace（ドキュメント・スプレッドシート）と連携 | テキスト生成・フォーム作成 |

社内がMicrosoft 365環境であればCopilotが使いやすく、Google Workspaceを使っているならGeminiが自然に連携できます。
どれが正解かは環境次第ですが、**無料プランで試してから判断する**ことをお勧めします。

### 教材の「型」を先に決める（OJT計画書 / テキスト / チェックリスト / クイズ）

**どの形式の教材を作るかを先に決めることで、AIへの指示が明確になり、生成物の精度が高まります。**
型が曖昧なまま指示すると、AIも曖昧な出力をしやすくなります。

代表的な教材の型と用途は以下の通りです。

- **OJT計画書**：研修スケジュール・担当者・評価基準を一覧化した管理用ドキュメント
- **テキスト（説明資料）**：業務内容・背景・ルールを文章で説明するメインコンテンツ
- **チェックリスト**：作業のステップや確認事項を箇条書きにした実務ツール
- **確認クイズ**：理解度を測るO×問題・選択問題のセット

この4種の組み合わせで、多くのOJT教材は構成できます。
まず「今回作るのはどの型か」を決めてからAIに指示することが、品質向上の鍵になります。

### 社内情報・機密データの取り扱いルールを確認する

**AIツールに社内情報を入力する前に、情報セキュリティポリシーを必ず確認してください。**
多くの無料プランのAIツールでは、入力したテキストがサービス改善に利用される場合があります。

入力を避けるべき情報の例を挙げます。

- 顧客名・取引先名・個人名
- 製品の製造プロセス・独自技術に関わる詳細情報
- 社内の数値情報（不良率・生産量・原価など）
- 従業員の個人情報

安全な運用のためには、固有名詞を「〇〇工程」「担当者A」のように匿名化・一般化して入力することをお勧めします。
企業向けプラン（ChatGPT Business・Copilot for Microsoft 365等）では入力データの学習利用が制限される場合があります。
本格導入を検討する際は、法人向けプランの活用も選択肢の一つです。

## 実践ステップ：OJT教材セットをAIで作る4ステップ

**ChatGPTを使ってOJT教材のたたき台を作るには、4つのステップを順番に進めると効率的です。**
ステップを分けることで、各段階の出力を確認・修正しながら進めることができます。

### ステップ1｜教材の骨格（目次・章立て）をAIに出力させる

**最初に目次・章立てをAIに作らせることで、教材の全体像を素早く確認できます。**
骨格が決まれば、その後の本文生成が格段に進めやすくなります。

まずは以下のような指示（プロンプト）を試してみてください。

```
あなたは製造業の新人教育担当者のアシスタントです。
以下の条件でOJT教材の目次を作成してください。

【対象者】製造ライン配属の新入社員（社会人経験なし）
【目的】入社後1週間で基本的な安全ルールと作業の流れを理解させる
【形式】全4章構成・各章3〜5項目
【含めるべき内容】安全装備の着用、KY活動（危険予知活動）、作業手順の確認方法、報告・連絡・相談のルール

目次を出力した後、各章の狙いを1〜2文で説明してください。
```

出力された目次を見て「足りない項目」「不要な項目」を調整することで、ゼロから構成を考える時間を削減できます。

### ステップ2｜各セクションの本文をAIで肉付けする

**目次が決まったら、章ごとに本文のたたき台を作成します。**
一度にすべての章を生成しようとすると出力が粗くなりやすいため、1章ずつ指示するのがお勧めです。

プロンプトの例は以下の通りです。

```
先ほど作成した目次の「第1章：安全装備の着用」の本文を作成してください。

【対象者】製造ライン配属の新入社員
【文章スタイル】です・ます調、専門用語には括弧で説明を加える
【長さ】A4用紙1枚程度（400〜600字）
【含めるべき内容】保護具の種類（安全帽・安全靴・保護メガネ）、着用タイミング、未着用時のリスク
【注意】数値・法令基準は「（※社内規定を確認のこと）」と注記してください
```

この「（※社内規定を確認のこと）」の注記指示が重要です。
AIが架空の数値や規定を断定的に書いてしまうことを防ぎ、後でレビューすべき箇所を明示できます。

> 関連記事：[ChatGPTで作業手順書・マニュアルを作成する方法](/chatgpt-sakugyotejunsho-manual-seizogyo)

### ステップ3｜チェックリスト・確認クイズをAIで追加する

**本文テキストができたら、理解度確認用のチェックリストとクイズをAIに作らせると、教材としての完成度が高まります。**
教えた内容を「実際にできるか確認する仕組み」をセットで用意することが、OJT教材の質を左右します。

チェックリスト生成の例は以下の通りです。

```
以下の内容をもとに、新入社員が自己チェックできる作業前確認チェックリストを作成してください。
チェック項目は10〜15個、「□ 〜している」の形式で書いてください。

【対象業務】製造ライン作業開始前の安全確認
【含める観点】保護具着用、機械の安全確認、作業エリアの整理整頓、異常時の報告先確認
```

確認クイズ生成の例は以下の通りです。

```
「製造ライン安全教育テキスト（第1〜3章）」の内容をもとに、
O×形式の確認問題を5問作成してください。
各問題に「正解」と「解説（2〜3文）」を添えてください。
難易度は「OJT1週間目の新入社員が答えられるレベル」にしてください。
```

チェックリストとクイズを揃えることで、「教えた→確認した」というOJTのサイクルが一つのセットとして完成します。

### ステップ4｜外国人・中途採用向けに「やさしい日本語」へ変換する

**外国人労働者や日本語に不慣れな中途採用者向けには、「やさしい日本語」への変換がAIの得意な作業の一つです。**
「やさしい日本語」とは、複雑な表現や漢字を減らし、誰でも理解しやすい形に書き換えた文体のことを指します。

変換プロンプトの例は以下の通りです。

```
以下の文章を「やさしい日本語」に書き換えてください。

【ルール】
- 1文を20〜30字程度に短くする
- 難しい漢字にはひらがなでルビを振る表記（例：安全（あんぜん））を加える
- カタカナ外来語は避け、わかりやすい日本語に置き換える
- 受動態（〜される）は能動態（〜します）に変える

【変換する文章】
（ここに先ほど作成した本文を貼り付ける）
```

やさしい日本語への変換は、外国人スタッフだけでなく、製造業での就業経験が少ない若手・中途採用者にとっても読みやすさが改善される可能性があります。
このステップを加えることで、より幅広い受講者に対応した教材を準備できます。

## コピペOKプロンプト集（業務場面別3種）

**実務で即使えるプロンプトを3種類まとめました。業務内容に合わせて括弧内を書き換えてお使いください。**

> 関連記事：プロンプトの書き方を基礎から学びたい方は [プロンプトの書き方入門【非IT職向け】](/prompt-kakikata-nyumon-hi-it-shoku) もご参照ください。

### 安全教育・KY活動向けプロンプト

```
あなたは製造業の安全教育担当アシスタントです。
以下の条件でKY活動（危険予知活動）の教材を作成してください。

【対象者】製造ライン配属 新入社員（入社1〜2週間目）
【業務場面】（例：プレス機周辺の作業 / フォークリフト通行エリアでの歩行 など）
【教材形式】
  ① KY活動の目的説明（200字程度）
  ② 場面別の危険ポイント（3〜5項目、箇条書き）
  ③ ヒヤリハット報告書の書き方（記入例付き）
  ④ 確認O×クイズ（3問、解説付き）

【注意事項】
- 法令・数値基準は「（※安全管理者に確認）」と記載する
- 断定的な表現は避け「〜する必要があります」「〜が重要です」の表現を使う
- やさしい言葉で書く（入社直後の新人でも読めるレベル）

※ 生成後、必ず現場の安全管理者がレビューしてください。
※ フォークリフト・クレーン・有機溶剤など、労働安全衛生法に基づく特別教育が必要な業務については、AI生成の教材をそのまま法定教育の教材として使用することはできません。必ず所管の法令・資格要件を確認の上、有資格の講師・専門機関にご相談ください。
```

### 作業手順・品質チェック手順向けプロンプト

```
あなたは製造ラインの作業手順書作成アシスタントです。
以下の条件で作業手順チェックリストを作成してください。

【対象業務】（例：射出成形品の外観検査 / 溶接後の仕上げ工程 など）
【対象者】OJT中の新入社員・中途採用者
【出力形式】
  ① 作業の目的と品質基準の概要（100〜150字）
  ② 作業前の準備チェックリスト（5〜8項目、□形式）
  ③ 作業手順（番号付きステップ形式、各ステップ1〜2文）
  ④ 品質確認チェックリスト（5〜10項目、□形式）
  ⑤ 異常発生時の対応フロー（3〜5ステップ）

【留意事項】
- 数値（トルク・温度・寸法など）は「（※作業標準書を確認）」と記載する
- 使用する工具・設備名は一般名称とし、機種名は「（社内設備名称を記入）」とする
- 1文は40字以内を目安に短く書く

※ 生成後、必ず現場リーダー・品質担当者がレビューしてください。
```

### 事務職向け業務フロー説明資料向けプロンプト

```
あなたは事務職の業務引き継ぎ・OJT資料作成アシスタントです。
以下の条件で業務フロー説明資料を作成してください。

【対象業務】（例：受注入力〜納品書発行の流れ / 月次経費精算の手順 など）
【対象者】事務職の新入社員・中途採用者（当該業務未経験）
【出力形式】
  ① 業務の目的と重要性（100字程度）
  ② 業務フロー図（テキストで表現：「ステップ1 → ステップ2 → ...」形式）
  ③ 各ステップの詳細説明（各ステップ3〜5文）
  ④ よくあるミスとその対処法（3〜5例）
  ⑤ 確認クイズ（選択肢4択、3問、解説付き）
```

## 注意点：AI生成教材を現場に出す前に必ず確認すること

**AIが生成した教材を、レビューなしでそのまま現場で使うことは避けてください。**
特に安全・法令・品質基準に関わる教材は、誤った情報が事故や法令違反につながるリスクがあります。

> **免責事項：** 安全教育・法令関連の教材はAI生成のまま使用しないでください。
> 必ず現場責任者・安全管理者が内容を確認・修正した上で使用してください。
> 本記事で紹介するプロンプトや手法は、教材作成の補助を目的としたものであり、
> 生成物の正確性・安全性を保証するものではありません。

### 数値・法令基準・社内規定の事実確認は必須

**AIは誤った数値や存在しない法令を自信満々に生成する場合があります。**
これは「ハルシネーション（幻覚）」と呼ばれる現象で、現在のAIが抱える根本的な課題です。

具体的に確認が必要な項目を挙げます。

- 労働安全衛生法関連の基準値（照度・換気量・騒音レベルなど）
- 機械設備の点検基準・資格要件
- 社内の承認フロー・報告ルール・権限規定
- 製品の品質基準・検査規格

これらの情報は、AIの出力を「参考の構成」として使いつつ、必ず一次情報（法令原文・社内規定書）と照合してください。

### 機密情報・個人情報の漏洩リスクと対策

**社内の機密情報をAIに入力する際は、情報漏洩のリスクを認識した上で判断してください。**
特に無料プランのAIサービスでは、入力データがモデルの学習に使われる場合があります。

安全な入力のための4つのルールを守ることをお勧めします。

1. **固有名詞を匿名化する**：「A社製品の〇〇工程」→「製品Xの△△工程」
2. **数値を一般化する**：「不良率〇〇%」→「不良率○%（実数は社内データに差し替える）」
3. **個人名を使わない**：「田中さんが担当する〇〇業務」→「担当者が行う〇〇業務」
4. **社内システム名を伏せる**：「〇〇ERPシステム」→「社内管理システム」

企業での本格運用を検討する場合は、入力データの学習利用が制限された法人向けプランの活用を検討することをお勧めします。

### 現場経験者のレビューを省略してはいけない理由

**AI生成教材には「現場の暗黙知」が欠けています。**
AIはテキスト上の情報から文章を生成するため、「この機械は音が変わったら止める」「この部品は力加減が大事」といった経験知を自動的に加えることはできません。

現場経験者のレビューを組み込むことで、以下の価値が生まれます。

- 実際の現場に即した注意事項・補足の追加
- 新人がつまずきやすいポイントの反映
- 「読んではわかるが実際はこうする」といった補足説明の追記

レビューは「内容の正誤確認」だけでなく「現場知識の補完」という目的で行うことが重要です。

### この記事の要点

- **役割分担を明確に**：AIが作る「たたき台」に、人間が「現場知識・事実確認」を加える
